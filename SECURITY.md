# Security Policy

## Reporting a Vulnerability

This repository was created using [Atlantis Starter #02](https://github.com/63klabs/atlantis-starter-02-apigw-lambda-cache-data-nodejs/) as a template.

If a security vulnerability is found, please report under the [Security menu in the serverless-remote-falcon-api-proxy GitHub repository](https://github.com/chadkluck/serverless-remote-falcon-api-proxy/security/advisories).
